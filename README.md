

# 说明

>  非常简单初级的一个vue demo，整个流程一目了然，适合入门。

>  项目源文件放在app文件夹下

## 项目运行
``` bash
# 克隆到本地
git clone git@github.com:olivianate/store.git

# 安装依赖
npm install

# 开启本地服务器
npm run dev

# 发布环境
npm run build
```
